# Riftbound TCG — Research Notes

**Source:** https://riftbound.leagueoflegends.com/en-us/

## What Is Riftbound?

A physical collectible trading card game by **Riot Games** (co-published with UVS Games), set in the **League of Legends universe (Runeterra)**. Designed by **Dave Guskin** (previously director of Legends of Runeterra).

- China launch: August 1, 2025
- Global launch: October 31, 2025
- Official description: *"Riftbound is a collectable trading card game featuring a vast array of characters and locales from League of Legends."*

---

## The Universe — Runeterra

Set in Runeterra, the shared LoL universe. Regions each have a distinct strategic identity:

| Region | Strategy |
|---|---|
| Demacia | Tempo, curve pressure |
| Noxus | Aggression, direct damage, burn |
| Ionia | Tricks, evasion, flash-speed interaction |
| Freljord | Ramp, resilience, oversized units |
| Piltover & Zaun | Combo, card advantage, draw engines |
| Bilgewater | Featured via Miss Fortune, Pyke |
| Shadow Isles | Featured via Karthus |
| Targon | Featured via Leona, Diana |
| The Void | Featured via Kai'Sa, Kha'Zix |

## The Six Domains (Core Faction System)

Every deck is defined by a **Champion Legend** card with **two Domains**:

| Domain | Color | Playstyle |
|---|---|---|
| **Fury** | Red | Aggression, discard, on-attack triggers |
| **Calm** | Green | Defensive, move effects, counter-tricks |
| **Mind** | Blue | Planning, Hidden mechanic, card draw, gear |
| **Body** | Orange | Ramp, unit buffs, Ready ability |
| **Chaos** | Purple | Recursion, unpredictability, discard both players |
| **Order** | White | Token swarms, Deathknell, go-wide aggression |

## Gameplay Mechanics

- **Win condition:** First to **8 points** (11 in team games)
- **Points scored by:** Conquering or Holding Battlefields
- **Resources:** 12-card Rune Deck (exhaust or spend runes)
- **Might:** Single stat = attack + health
- **Deck:** 40-card main deck + 12 rune cards + 1 Champion Legend
- **Formats:** 1v1, 2v2, FFA (up to 4 players); ~10–20 min games

### Key Keywords / Mechanics
- **Accelerate / Ready** — enters ready (Body)
- **Hidden** — surprise plays (Mind, Chaos)
- **Deathknell** — value from dying units (Order)
- **Equipment (Gear)** — persists after unit death (Spiritforged)
- **Gold** — flexible power tokens (Spiritforged — Draven/Renata)
- **Repeat** — pay spell cost twice to double effect (Spiritforged)
- **Ambush** — play unit as Reaction to a battlefield (Unleashed)
- **XP** — champion leveling/growth (Unleashed — Kha'Zix)
- **Hunt** — (Unleashed)

### Card Rarities
Common → Uncommon → Rare → Epic → **Overnumbered** (secret ultra-rare)
- Rare, Epic, Alt Art, and Overnumbered are always foil
- **Ultimate Rarity** introduced in Set 3 (Baron Nashor — first Ultimate Rare)

## Sets / Expansions

### Set 1: Origins (Oct 31, 2025)
- 377 cards, ~40 champions
- 12 Champion Legends: **Kai'Sa, Volibear, Jinx, Darius, Ahri, Lee Sin, Yasuo, Leona, Teemo, Viktor, Miss Fortune, Sett**
- Starter box: Proving Grounds ($29.99) — Annie, Master Yi, Lux, Garen decks

### Set 2: Spiritforged (Feb 13, 2026)
- 200+ new cards
- Theme: "Unites the power of forged steel with the ethereal presence of the Ionia Spirit Blossom"
- 12 new Legends: **Rek'Sai, Draven, Irelia, Rumble, Lucian, Sivir, Azir, Ezreal, Ornn, Renata Glasc, Fiora, Jax**
- New mechanics: Equipment, Gold, Repeat

### Set 3: Unleashed (May 8, 2026 — global)
- 220+ cards
- Set tagline: **"Power strikes from the shadows."**
- Theme: champions whose power was earned in darkness/solitude — "Real strength doesn't announce itself — it lurks, builds, and erupts when the moment is right."
- 4 new Legends: **Kha'Zix, Lillia, Diana, Ivern**
- Also: Vi, Vex, Rengar, Poppy, LeBlanc, Pyke
- New mechanics: Ambush, XP, Hunt
- **Baron Nashor** = first Ultimate Rare card

### Set 4: Vendettas (Q3 2026)
- New Legends: **Akali, Mel, Ambessa**
- First simultaneous global release

## Key Champions

- **Ahri** — Nine-Tailed Fox; reduces enemy Might when attacking controlled battlefield
- **Darius** — Executioner variant; gains power when multiple cards played in a turn
- **Draven** — Showboat form; power scales with player's point total
- **Ekko** — Can be recycled from trash after death (time-manipulation lore)
- **Fiora** — At 5 Might unlocks movement, deflect, and shield abilities
- **Heimerdinger** — Inherits all abilities of friendly legends/units/gears on board
- **Viktor** — Earns recruit tokens when any unit dies
- **Volibear** — Costs 10 energy; splits 5 damage between enemies when attacking
- **Kha'Zix** — Gains 1 XP per combat win; unlocks new abilities at each XP level
- **Jinx** — Fury/Chaos domain pairing
