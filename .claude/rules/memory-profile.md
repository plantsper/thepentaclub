# Profile

## Project
Building **The Pentaclub** — a Riftbound TCG card catalog SPA for the Riftbound (Riot Games) trading card game.
- Solo developer and sole admin
- Deployed on Vercel (static), Supabase backend
- Custom admin CMS with image upload, Claude vision OCR, and Riftcodex API integration

## Game Domain Knowledge
Riftbound is a physical TCG by Riot Games set in the League of Legends universe (Runeterra).
- Card sets identified by short code: `OGN` (Origins), `SFD` (Spiritforged), etc.
- Card rarities: Common, Uncommon, Rare, Epic, Overnumbered, Ultimate
- Card variants: standard (`170/221`), overnumber (`100/99`), alt-art (`000a/100`), signature (`200*/199`)
- Stat terminology: Might = single combined power/health stat (though the site uses Power/Health separately)
